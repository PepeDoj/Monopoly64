# Monopoly64

Mopoly64 - версия знаменитой игры моноплия выполненная в консольном виде при помощи c#.

## Механики игры
+ Партия от 2-3 игроков
+ Карточки событий
+ Покупка недвижимости
+ Уплата налогов и получение зарплаты
+ Лишь 1 победитель
+ Все правила игры описаны на игровом поле
